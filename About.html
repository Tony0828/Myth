About Me

Hi, my name is Tony. I am a tenth grader attending Ridley College. I made this website for my personal project. I am interested in science and curious about programming.

Why

I made this website because I wanted to encorporate both my interest in science and curiosity for programming. I see that many people still believe in many myths, so I wanted to help inform people about these myths. Before doing this project I believed in some of these myths so I think that it is getting more and more important for people to know the truth, so hopefully after visiting this website you will understand more common myths as well as avoid them more often.

Reliability

All of the information provided on this website has been sourced from reliable sources and will be included in the Sources section in MLA format if you would like to check it out.
